# Name of my project: B13-A7-keen-keeper

A short description:

Keen Keeper is a modern React-based application designed for managing relationships and tracking interactions. It allows users to monitor their connections, record different types of interactions (Call, Text, Video), and gain insights through visual analytics and an interactive timeline.

Technologies used:
React.js
React Router
Context API
Tailwind CSS
Recharts
React Toastify
Vite

3 key features of project:
1. Interaction Logging System

Users can easily record interactions such as calls, texts, and video chats with their friends, all stored in a centralized system for quick access.

2. Real-Time Timeline View

The app presents all activities in a dynamic timeline, allowing users to browse, search, and filter interactions effortlessly.

3. Visual Analytics Dashboard

An interactive dashboard displays interaction statistics using a pie chart, helping users understand communication patterns at a glance.
